{% comment %} Image template. Args: img (required) - path relative to site root, e.g. coding/img/foo.png title (optional) - tooltip and link title alt (optional) - alt text; falls back to title caption (optional) - figcaption (renders as italic, muted, centered) url (optional) - if present, image is wrapped in an width (optional) - one of sm | md | lg (default) | xl | full align (optional) - one of left | center (default) | right Backward-compatible with old callsites that only set img/title/caption/url. {% endcomment %} {%- assign _alt = include.alt | default: include.title -%} {%- case include.width -%} {%- when 'sm' -%} {%- assign _maxw = 'max-w-sm' -%} {%- when 'md' -%} {%- assign _maxw = 'max-w-md' -%} {%- when 'xl' -%} {%- assign _maxw = 'max-w-3xl' -%} {%- when 'full' -%} {%- assign _maxw = 'max-w-full' -%} {%- else -%} {%- assign _maxw = 'max-w-2xl' -%} {%- endcase -%} {%- case include.align -%} {%- when 'left' -%} {%- assign _align = 'mr-auto ml-0 text-left' -%} {%- when 'right' -%} {%- assign _align = 'ml-auto mr-0 text-right' -%} {%- else -%} {%- assign _align = 'mx-auto text-center' -%} {%- endcase -%}
